# PDF Maker

CLI-скрипт для генерации PDF-счёта из CSV. Читает список товаров из `data/products.csv`, формирует таблицу и сохраняет один PDF в папку `output/`. Поддерживает кириллицу (шрифт Arial из Windows).

## Структура проекта

```
pdf_maker/
├── data/
│   └── products.csv    # исходные данные (товар, цена, количество)
├── output/
│   └── invoice.pdf     # сгенерированный счёт
├── main.py             # точка входа
├── requirements.txt
└── README.md
```

## Требования

- Python 3.9+
- Windows (для шрифта Arial и автоматического открытия PDF)

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

Скрипт создаёт `output/invoice.pdf` с таблицей по всем строкам из CSV и по завершении открывает его в программе просмотра по умолчанию.

## Формат CSV

Файл `data/products.csv` должен содержать заголовки и данные в кодировке UTF-8:

| Столбец   | Описание    |
|-----------|-------------|
| `product` | Название товара |
| `price`   | Цена за единицу |
| `qty`     | Количество |

Пример:

```csv
product,price,qty
Ноутбук Dell,85000,5
Клавиатура Logitech,3500,12
Мышь Razer,2500,8
```

Сумма по строке считается автоматически: `price × qty`.

## Обработка ошибок

- Если нет файла `data/products.csv` или в нём нет столбца `product` — скрипт выводит сообщение об ошибке и завершается.
- Папка `output/` создаётся автоматически при первом запуске.
- Шрифт Arial берётся из `C:\Windows\Fonts\arial.ttf`; при его отсутствии выводится предупреждение.
